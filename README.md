# 🏨 Hotel Management System

A console-based Hotel Management System built using Python.

## 🚀 Features
- Room Booking
- Food Ordering
- Activity Management
- Billing System
- CSV Data Storage
- Graph Visualization

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib

## ▶️ How to Run
```bash
pip install pandas numpy matplotlib
python main.py
